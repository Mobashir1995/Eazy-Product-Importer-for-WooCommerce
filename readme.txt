Easy Product Importer for WooCommerce helps to import all types of products from one store to another store easily

== Description ==
You can easily import any types of products from one WooCommerce store to another WooCommerce store easily. It will help you to import unlimited numbers of products. Moreover, you can import all types of product data such as Image Gallery, Variations, Downloadable files, Attributes, Categories etc. You need to keep the WooCommerce Plugin installed and activated on your website to work with this plugin.

== Frequently Asked Questions ==
= Does this plugin work without WooCommerce? =
This plugin needs WooCommerce installed and activated.

= Does this plugin import other plugins product without WooCommerce =
This plugin only import WooCommerce products
